# Operator Guide

## First Deployment

1. Define collection points and material categories.
2. Register operators and custody checkpoints.
3. Record first collection batch.
4. Run grade and routing recommendations in assisted mode.
5. Produce a monthly impact report.

## Minimum Production Controls

- scale or count evidence
- custody transfer log
- contamination and rejection reason codes
- buyer specification records
- report reconciliation

## Certification

Certified operators must prove traceability from collection through reuse,
sale, recycling or disposal.
